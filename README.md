# Proyecto Cursor IA con SQL Server

Este proyecto consiste en una aplicación desarrollada en Python con apoyo
de Cursor IA, la cual permite la interacción con el usuario y el
almacenamiento de datos en una base de datos SQL Server.

## Tecnologías utilizadas
- Cursor IA
- Python
- SQL Server
- pyodbc

## Ejecución
```bash
python main.py
